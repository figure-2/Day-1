# Linux comand

>리눅스 명령어덜을 정리해봅시다

- 'pwd' (print working directory)
    - 현재 작업중인 경로를 출력
- 'is' (list)
    - 현재 폴더에 있는 파일, 폴더를 출력
    출력
    - -a : 숨김처리된 파일, 폴더까지 출력
- 'cd' ( change directory)
    - 폴더 이동

- 'mkdir' (make directory)
    - 폴더 생성

- 'touch'
    - 파일 생성
- 'rm' (remove)
    - 삭제하는 명령어
    '-r' (optional)
